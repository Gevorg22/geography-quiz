# Geography Quiz

Интерактивная географическая викторина. На экране вращается 3D-глобус, приложение называет страну на русском языке, а игрок должен найти и кликнуть её на карте.

Демо: https://gevorg22.github.io/geography-quiz/

## Как играть

На каждый вопрос даётся три попытки.

Если страна угадана с первой попытки, она подсвечивается зелёным и начисляется 3 очка.

Если со второй, желтым и 1 очко.

Если правильный ответ не найден за три клика, страна помечается красным и очки не начисляются.

Кнопка "Не знаю / Пропустить" позволяет перейти к следующему вопросу без штрафа.

Кнопка "Завершить" заканчивает текущую сессию и показывает итоговую статистику.

## Возможности

158 стран в базе (все, представленные в TopoJSON).

Случайный порядок вопросов в каждой сессии.

Два режима игры: классика (найти страну по названию) и столицы (найти страну по столице).

Фильтр по региону: весь мир, Европа, Азия, Африка, Америка, Океания.

Фильтр по сложности: лёгкий, средний, сложный.

Streak-бонус: серия правильных ответов с первой попытки отображается огнём 🔥.

Подсказка с вращением глобуса к нужной стране (даётся один раз за вопрос с штрафом к очкам).

Итоговая статистика: количество угаданных с первой, второй, третьей попытки и пропущенных, суммарный счёт, рейтинг.

История игр и топ рекордов в localStorage.

Раздел слабых мест: страны, которые чаще всего не угадываются.

Звуковые эффекты и тактильный отклик (haptic feedback) на мобильных устройствах.

Вращение глобуса мышью и пальцем на мобильных устройствах.

Масштабирование колесом мыши и кнопками +/−.

Анимация ripple при касании глобуса.

Адаптивный дизайн для мобильных устройств.

Установка как PWA через кнопку "Добавить на экран домой".

## Мобильные улучшения

Уведомление о правильном или неправильном ответе всегда отображается по центру видимой части экрана независимо от уровня приближения.

Название искомой страны всегда остаётся в зоне видимости при pinch-zoom.

Заголовок с прогрессом, очками и кнопкой "Завершить" привязан к visual viewport и остаётся видимым при любом уровне приближения.

Кнопки масштабирования +/− привязаны к visual viewport и всегда доступны при pinch-zoom.

При приближении через тачскрин все UI-элементы сохраняют свой визуальный размер (counter-scale через `1 / vp.scale`).

## Тесты

Проект покрыт тестами на базе **Vitest** + **React Testing Library** (78 тестов, 9 файлов).

| Файл | Что проверяется |
|---|---|
| `hooks/useGame.test.ts` | Игровая логика: угадывание, ошибки, пропуск, рестарт, статистика |
| `utils/array.test.ts` | Перемешивание массива (Fisher–Yates) |
| `utils/time.test.ts` | Форматирование времени `мм:сс` |
| `components/FeedbackToast/` | Сообщения по типу тоста, CSS-модификаторы, aria-атрибуты, Visual Viewport hook (подписка, отписка, масштаб) |
| `components/TaskPanel/` | Название и язык страны, точки попыток, кнопка пропуска |
| `components/GameHeader/` | Логотип, прогресс-бар, очки, кнопка завершения |
| `components/ZoomControls/` | Кнопки +/−, колбэки, aria-метки |
| `components/GameOver/` | Статистика попыток, рейтинг, итоговый счёт, кнопка рестарта |

## Структура проекта

    src/
    constants/     константы игры, цвета, очки, зум
    data/          список стран с id, названиями, регионом, сложностью, столицами и фактами
    hooks/         useGame, useHistory, useVisualViewport
    utils/         shuffle, sounds, haptic
    components/
      Globe/         D3-глобус с анимацией и ripple-эффектом
      GameHeader/    шапка с прогрессом и очками (привязана к visual viewport)
      TaskPanel/     текущий вопрос и попытки (привязан к visual viewport)
      ZoomControls/  кнопки масштабирования (привязаны к visual viewport)
      FeedbackToast/ уведомления об исходе хода (привязаны к visual viewport)
      GameOver/      экран итоговой статистики и рекордов
      GameSetup/     экран настроек перед началом игры

## Деплой на GitHub Pages

В репозитории уже настроен GitHub Actions workflow (.github/workflows/deploy.yml).

После пуша в ветку main он автоматически собирает проект и деплоит на GitHub Pages.

Для активации GitHub Pages перейдите в Settings репозитория, раздел Pages, источник выберите "GitHub Actions".

Приложение будет доступно по адресу https://gevorg22.github.io/geography-quiz/
