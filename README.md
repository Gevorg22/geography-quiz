# Geography Quiz

Интерактивная географическая викторина. На экране вращается 3D-глобус, приложение называет страну на русском языке, а игрок должен найти и кликнуть её на карте.

Демо: https://gevorg22.github.io/geography-quiz/

## Как играть

На каждый вопрос даётся три попытки.

Если страна угадана с первой попытки, она подсвечивается зелёным и начисляется 3 очка.

Если со второй, желтым и 1 очко.

Если правильный ответ не найден за три клика, страна помечается красным и очки не начисляются.

Кнопка "Не знаю / Пропустить" позволяет перейти к следующему вопросу без штрафа.

Кнопка "Завершить" заканчивает текущую сессию и показывает итоговую статистику.

## Возможности

158 стран в базе (все, представленные в TopoJSON).

Случайный порядок вопросов в каждой сессии.

Счётчик времени.

Итоговая статистика: количество угаданных с первой, второй, третьей попытки и пропущенных, суммарный счёт, общее время, рейтинг.

Вращение глобуса мышью и пальцем на мобильных устройствах.

Масштабирование колесом мыши и кнопками +/-.

Адаптивный дизайн для мобильных устройств.

## Тесты

Проект покрыт тестами на базе **Vitest** + **React Testing Library** (78 тестов, 9 файлов).

| Файл | Что проверяется |
|---|---|
| `hooks/useGame.test.ts` | Игровая логика: угадывание, ошибки, пропуск, рестарт, статистика |
| `hooks/useTimer.test.ts` | Секундомер: старт, stop, reset, инкремент |
| `utils/array.test.ts` | Перемешивание массива (Fisher–Yates) |
| `utils/time.test.ts` | Форматирование времени `мм:сс` |
| `components/FeedbackToast/` | Сообщения по типу тоста, CSS-модификаторы, aria-атрибуты, Visual Viewport hook (подписка, отписка, масштаб) |
| `components/TaskPanel/` | Название и язык страны, точки попыток, кнопка пропуска |
| `components/GameHeader/` | Логотип, прогресс-бар, таймер, очки, кнопка завершения |
| `components/ZoomControls/` | Кнопки +/−, колбэки, aria-метки |
| `components/GameOver/` | Статистика попыток, рейтинг, итоговый счёт, кнопка рестарта |

## Структура проекта

    src/
    constants/     константы игры, цвета, очки, зум
    data/          список стран с id и названиями
    hooks/         useGame, useTimer
    utils/         formatTime, shuffle
    components/
      Globe/         D3-глобус с анимацией
      GameHeader/    шапка с прогрессом и таймером
      TaskPanel/     текущий вопрос и попытки
      ZoomControls/  кнопки масштабирования
      FeedbackToast/ уведомления об исходе хода
      GameOver/      экран итоговой статистики

## Деплой на GitHub Pages

В репозитории уже настроен GitHub Actions workflow (.github/workflows/deploy.yml).

После пуша в ветку main он автоматически собирает проект и деплоит на GitHub Pages.

Для активации GitHub Pages перейдите в Settings репозитория, раздел Pages, источник выберите "GitHub Actions".

Приложение будет доступно по адресу https://gevorg22.github.io/geography-quiz/
